## üìå Project Overview
Handwritten medical documents such as prescriptions, clinical notes, and diagnostic reports are widely used in healthcare. However, variations in handwriting styles, unclear text, and unstructured formats make these documents difficult to read and process manually.

This project implements an **AI-powered Handwritten Medical Transcript Summarization System** that converts handwritten medical documents into digital text using **Optical Character Recognition (OCR)** and generates **concise, structured summaries** using **Natural Language Processing (NLP)**. The system improves efficiency, accuracy, and accessibility of medical records.

## üéØ Objectives
- Automate extraction of handwritten medical text using OCR  
- Generate meaningful summaries using NLP techniques  
- Reduce manual transcription errors and workload  
- Improve readability and accessibility of medical records  
- Provide a secure and user-friendly web application
